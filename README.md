JeePS


Overview

JeePS is a mobile application designed to help commuters navigate jeepney routes in the Philippines. It focuses on making public transport more accessible and less confusing for everyone.


Problem Statement

Commuting can be difficult for many people. Common issues include not knowing which jeepney to ride, being unsure of the fare, and being unfamiliar with the route. These problems often cause people to avoid jeepneys even though they are an affordable transport option.


Our Solution

The JeePS app recommends the best jeepney routes based on the current location and destination of the user. It provides clear details such as the estimated fare and important landmarks. A key feature is showing the actual signboard text of the jeepney to help users identify the right vehicle in real time.


Key Features

• Location Access: Detects the current position of the user to find nearby routes.

• Route Search: Allows users to enter their destination to see possible jeepney paths.

• Fare Estimation: Calculates costs for regular and discounted passengers.

• Signboard Display: Shows the specific text commuters should look for on the vehicle.

• Route Details: Provides a list of stops and landmarks for better guidance.


Tech Stack

• Language: Kotlin

• UI Framework: Jetpack Compose

• Maps and Location: Google Maps SDK

• Search and Autocomplete: Google Places API

• Backend and Database: Supabase


Innovation and Impact

JeePS provides localized guidance that many standard navigation apps do not offer. It addresses specific needs like fare information and visual signboard identification. The project supports Sustainable Development Goal 11 for Sustainable Cities and Communities and Goal 9 for Industry, Innovation and Infrastructure.

Current Coverage

The application currently covers the corridor from San Pedro, Laguna to Calamba, Laguna. The system is designed to scale and support more routes in other cities and provinces in the future.
